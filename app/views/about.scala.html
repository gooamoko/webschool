Проект Web Archive создан для обеспечения возможности заполнения бланочной продукции Прокопьевского горнотехнического техникума им. В.П.Романова. Проект ориентирован на быстрое добавление студентов, которые числятся в архиве. Именно поэтому он называется веб архивом.

Почему Web-приложение?

Реализация архива в виде web-приложения имеет свои плюсы, а именно:

  • Для работы с приложением клиенту не требуется ничего кроме браузера;
  • Проект проще обновлять, поскольку это делается централизованно;
  • Для прекращения работы с проектом сразу всех пользователей (если это необходимо) достаточно остановить сервер;

На чем реализовано это приложение?

Это приложение создано на языке программирования Java с использованием Play Framework.
